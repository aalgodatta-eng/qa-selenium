Jenkins forgot password
Step 1 — Stop Jenkins
docker compose -f docker-compose-jenkins.yml down
Step 2 — Remove Jenkins volume

Check volume name:

docker volume ls

You will see something like:

kc-sdet-automation_jenkins_home

Remove it:

docker volume rm kc-sdet-automation_jenkins_home

(Use exact name from your system)

Step 3 — Start Jenkins again
docker compose -f docker-compose-jenkins.yml up -d --build

Now Jenkins will show:

Unlock Jenkins

Get initial password:

docker exec -it jenkins cat /var/jenkins_home/secrets/initialAdminPassword
winpty docker exec -it jenkins sh -lc "cat /var/jenkins_home/secrets/initialAdminPassword"

Copy that password → paste in browser.

✅ OPTION 2 — Just Reset Password (Without Deleting Data)

If you don’t want to lose jobs:

Step 1 — Enter Jenkins container
docker exec -it jenkins bash
Step 2 — Create new admin user via Groovy

Inside container:

cd /var/jenkins_home
vi init.groovy.d/reset.groovy

Paste this:

import jenkins.model.*
import hudson.security.*

def instance = Jenkins.getInstance()

def hudsonRealm = new HudsonPrivateSecurityRealm(false)
hudsonRealm.createAccount("admin", "admin123")
instance.setSecurityRealm(hudsonRealm)
instance.save()

Save and exit.

Step 3 — Restart container

Exit container and run:

docker restart jenkins

Now login with:

Username: admin
Password: admin123
